파일 업로드 → 등록된 용어 하이라이트

파일을 업로드하면 한국어 문자열을 추출해서 등록된 용어를 자동으로 하이라이트하고 영어 번역을 보여줍니다.
지원 형식: 엑셀(.xlsx, .xls), CSV, TSV, Word(.docx), JSON, XML, TXT, MD, HTML, PO, properties, strings, resx

엑셀 → 용어집 일괄 가져오기

기존 엑셀/CSV 용어집을 통째로 등록합니다. 한국어가 같은 용어가 이미 있으면 영어/예문을 덮어씁니다 (프로젝트 태그는 병합).
지원 형식: 엑셀(.xlsx, .xls), CSV, TSV

한영 작업 파일 → AI가 용어 후보 추출

한국어 원문과 영어 번역이 같이 있는 파일을 업로드하면 Claude AI가 용어 후보를 추출합니다. (Claude API 키 필요)
표 형식(엑셀/CSV/TSV)은 컬럼을 직접 선택하고, 그 외 형식(JSON/XML/TXT/PO 등)은 AI가 한/영 쌍을 자동으로 찾습니다.

파일이 더 크면 앞부분만 분석됩니다. 비용 관리용.

Claude API 키 미설정

AI 후보 추출 기능을 사용하려면 Anthropic Claude API 키가 필요합니다. console.anthropic.com에서 발급받을 수 있어요. 키는 본 브라우저에만 저장되며 외부로 전송되지 않습니다 (Anthropic 서버 제외).

정상 키는 sk-ant-api03-으로 시작하고 약 108자입니다. 보기 버튼으로 정확히 입력됐는지 확인하세요.

프로젝트 관리

게임/프로젝트별로 용어를 분류할 수 있습니다.

데이터 백업 / 복원

용어집 데이터를 JSON 파일로 내보내거나 가져올 수 있습니다. 팀 공유나 다른 PC로 옮길 때 사용하세요.
